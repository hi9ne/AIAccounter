Step 1: Deploy Backend to Railway

  1. Go to railway.app
  2. Create new project from GitHub
  3. Select this repository
  4. Set root directory: backend
  5. Add environment variables from backend/.env

Your backend URL will be something like: https://aiaccounter-production.up.railway.app

Step 2: Update Frontend Config

Edit miniapp/miniapp-config.js and replace:

'https://your-railway-app.up.railway.app/api/v1'

with your actual Railway URL

Step 3: Update Telegram Bot

  1. Open BotFather in Telegram
  2. Use /setmenubutton
  3. Select your bot
  4. Set URL to your Cloudflare Pages URL

Step 4: Test

Open your Telegram bot and click the menu button to launch the Mini App

Go to App (will only work after proper setup)
